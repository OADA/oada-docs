# Supported HTTP Status Codes

# Success

* **200** OK
* **201** Created
* **204** No Content
* **206** Partial Content

# Redirection

* **301** Moved Permanently
* **304** Not Modified
* **307** Temporary Redirect

# Client Error

* **400** Bad Request
* **401** Unauthorized
* **403** Forbidden
* **404** Not Found
* **406** Not Acceptable
* **409** Conflict
* **411** Length Required
* **412** Precondition Failed
* **415** Unsupported Media Type
* **416** Requested Range Not Satisfiable
* **429** Too Many Requests
